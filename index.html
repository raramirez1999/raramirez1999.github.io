Analyzing Zillow Rental Properties

In this data-driven project, I employed Python and a Zillow API to webscrape data from rental properties in San Diego County. Using data wrangling and cleaning techniques in Excel, I structured the data for exploratory data analysis. The insights gathered through this process culminated in the creation of an insightful dashboard, facilitating informed decision-making for a group of professionals seeking fairly priced housing options in San Diego.

Analyzing Chicago Communities Hardship

In this project, I utilized Python within Jupyter Lab to leverage the pandas library, converting a CSV file into a SQL dataset. Through SQL queries and the application of seaborn and matplotlib for data visualization, I conducted a comprehensive analysis. Drawing on my mathematical background and statistical analysis, I derived valuable insights on the rising hardship index in Chicago communities.

Regression Modeling Real Estate

Built a regression model to aid a Real Estate agent in accurately pricing and selling a home for $520,000 in an unfamiliar market. Utilized an API to web scrape sold listings in the surrounding neighborhoods, cleaned data using Python Pandas library and trained a Scikit-learn multiple linear regression model to predict pricing based on several parameters. Finally, matplotlib and seaborn for visuals.
